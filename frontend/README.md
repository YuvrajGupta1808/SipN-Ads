# Sip N'ads

Shipping options and delivery experience — manage preferences and get the best options for your shipments.

## Run locally

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:8080](http://localhost:8080).

## Scripts

- `npm run dev` — start dev server (port 8080)
- `npm run build` — production build
- `npm run preview` — preview production build
- `npm run lint` — run ESLint

## Tech

- Vite, React, TypeScript
- Tailwind CSS, shadcn/ui
- React Router, TanStack Query, Framer Motion

## Project structure

- `src/pages/` — Index (landing), Chat, NotFound
- `src/components/` — UI components and feature components
- `public/` — static assets (favicon, etc.)
